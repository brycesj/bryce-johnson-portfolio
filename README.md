# Bryce Johnson Portfolio

Next.js, React, and Tailwind CSS portfolio for Electrical Engineering projects.

## Edit Projects

Project content lives in `data/projects.ts`. Add future projects there, then place media in:

- `public/projects/line-following-robot/`
- `public/projects/spark-lab/`
- `public/projects/portable-console/`
- `public/projects/water-bottle/`
- `public/projects/speaker/`

Use descriptive filenames like `front-view.jpg`, `circuit-design.jpg`, `implementation.jpg`, or `demo.mp4`.

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
